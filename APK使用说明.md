## 📱 安装步骤

### 方法一：直接复制到手机安装

1. 将 `app-debug.apk` 文件复制到手机内存或SD卡
2. 在手机上打开文件管理器
3. 找到并点击 `app-debug.apk` 文件
4. 如果提示"禁止安装未知来源应用"：
   - 进入手机 **设置** → **安全**
   - 开启 **允许安装未知来源应用**
5. 按照提示完成安装

### 方法二：使用USB数据线安装

**电脑端操作：**
1. 用USB数据线连接手机和电脑
2. 在手机上选择"文件传输"模式
3. 将 `app-debug.apk` 复制到手机
4. 在手机上安装（步骤同上）

**或者使用ADB命令（需要开启USB调试）：**
```powershell
adb install e:\新建文件夹\KEBIAO\android\app\build\outputs\apk\debug\app-debug.apk
```

### 方法三：发送到手机

1. 在电脑上右键点击 `app-debug.apk`
2. 选择"发送到" → "蓝牙设备" 或 "微信" 等
3. 在手机上接收并安装

## ❓ 常见问题

### Q: 安装时提示"禁止安装"？
**A:** 
1. 进入手机 **设置** → **应用**
2. 找到浏览器或文件管理器
3. 开启 **允许安装未知来源应用**
4. 或者在安装时点击"设置" → 开启权限

### Q: 应用无法打开？
**A:** 
1. 检查Android版本是否 ≥ 7.0
2. 重新安装应用
3. 清除应用缓存后重试

### Q: 无法选择课表文件？
**A:** 
1. 确保课表文件格式为 .xls 或 .xlsx
2. 尝试使用文件管理器选择文件
3. 检查应用权限是否开启存储权限

### Q: 课程显示不完整？
**A:** 
1. 上传完整的课表文件
2. 检查Excel文件格式是否正确
3. 重新上传课表
